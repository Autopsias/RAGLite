================================================================================
DATA QUALITY NOTES
================================================================================

TRANSPOSED TABLES:
  - Table T3-002 (Headcount Evolution) has unusual structure
  - Months/years as COLUMNS, entities as ROWS
  - Standard parsers expecting entity headers in row 1 will fail
  - Requires special timeline-aware parsing

MULTI-LEVEL HEADERS:
  - Tables T3-003 (Capex) and T3-005 (Operational Costs)
  - Grouped headers and sub-headers
  - Careful cell-to-header mapping required

PERCENTAGE METRICS:
  - Budget Variance % (all tables)
  - Prior Year Variance %
  - Market Share %
  - Product/Fuel Mix %
  - Some with basis points: -0.5 pp

UNIT CONVERSIONS:
  - Volumes: km³ vs m³ (1 km³ = 1,000,000 m³)
  - Weight: kton vs ton (1 kton = 1,000 ton)
  - Currency: EUR (thousands) unless specified
  - Energy: Kcal/kg, Kwh/ton

================================================================================
SEMANTIC CHALLENGES
================================================================================

1. TRANSPOSED TABLE PARSING
   - Need to identify timeline-oriented tables
   - Handle month/year column headers differently
   - GTQ-3-002 tests this capability

2. NEGATIVE VALUE INTERPRETATION
   - Costs shown as negative (e.g., -77.1 EUR/m3)
   - Variance shown as negative percent (e.g., -37%)
   - GTQ-3-010, GTQ-3-011 test understanding

3. DERIVED METRICS
   - EBITDA calculated from price - variable cost - fixed cost
   - Unit margins require multi-row calculation
   - GTQ-3-009, GTQ-3-014 test derivation capability

4. CROSS-ENTITY AGGREGATION
   - Region totals from individual region data
   - Plant totals from individual plant data
   - GTQ-3-015 tests aggregation accuracy

5. BUDGET VARIANCE ANALYSIS
   - Percentage variance vs absolute amount
   - Prior year vs budget comparison
   - GTQ-3-003, GTQ-3-011 test interpretation

6. SCALE ECONOMICS
   - Unit cost variation across entities
   - Outão (798 kton) vs Pataias (44 kton)
   - Different EBITDA despite same pricing
   - Tests understanding of operational efficiency
